## System Overview

| Feature | SAYC Specification |
|---------|-------------------|
| 1NT Range | **15-17 HCP** |
| Major Requirement | **5 cards** |
| Minor Requirement | 3+ cards |
| Strong Opening | 2♣ (22+ HCP) |
| Weak Twos | 2♦/♥/♠ (5-11 HCP, 6 cards) |
| Origin | ACBL, 1980s |
| Popularity | Worldwide, BBO default |

### Design Philosophy

SAYC was created by the American Contract Bridge League to:
- Provide a standard system for "Yellow Card" events
- Eliminate alerts and misunderstandings in casual partnerships
- Serve as a baseline for online play (OKbridge, BBO)

## Opening Bids

### Summary Table

| Open | HCP | Hand Description |
|------|-----|------------------|
| 1♣ | 13-22 | 3+♣ (bid with 3-3 minors) |
| 1♦ | 13-22 | 4+♦ (bid with 4-4 minors), or 3♦ with 4-4-3-2 |
| 1♥ | 13-22 | **5+♥** |
| 1♠ | 13-22 | **5+♠** |
| 1NT | **15-17** | Balanced (may include 5-card major) |
| 2♣ | **22+** | Strong artificial, or 9+ playing tricks |
| 2♦ | 5-11 | 6+♦, weak two |
| 2♥ | 5-11 | 6+♥, weak two |
| 2♠ | 5-11 | 6+♠, weak two |
| 2NT | **20-21** | Balanced |
| 3♣/♦/♥/♠ | 5-11 | 7+ card suit, preemptive |
| 3NT | **25-27** | Balanced |
| 4♣/♦ | 5-11 | 8+ card suit, preemptive |
| 4♥/♠ | - | 7-8 playing tricks, preemptive |

### Opening Bid Rules

**Suit Selection with Equal Length:**
- **5-5 or longer:** Open the **higher-ranking** suit
- **4-4 in minors:** Open **1♦**
- **3-3 in minors:** Open **1♣**
- **4-4 in majors:** Cannot happen (need 5 to open major)

**Balanced Hand Ranges:**
| HCP | Action |
|-----|--------|
| 12-14 | Open 1 of suit, rebid 1NT |
| 15-17 | Open **1NT** |
| 18-19 | Open 1 of suit, jump rebid 2NT |
| 20-21 | Open **2NT** |
| 22-24 | Open **2♣**, rebid 2NT |
| 25-27 | Open **3NT** (or 2♣, rebid 3NT) |

## Responses to 1NT

Opening 1NT shows **15-17 HCP, balanced** (may include 5-card major or minor).

### Response Summary

| Response | HCP | Meaning |
|----------|-----|---------|
| PASS | 0-7 | No game interest |
| **2♣(!)** | 8+ | **Stayman** (non-forcing) |
| **2♦(!)** | 0+ | **Jacoby Transfer** to ♥ |
| **2♥(!)** | 0+ | **Jacoby Transfer** to ♠ |
| **2♠(!)** | 0+ | Transfer to 3♣ (weak minor) |
| 2NT | 8-9 | Invitational, balanced |
| 3♣/3♦ | 10-12 | 6+ cards, invitational, 2 of top 3 honors |
| 3♥/3♠ | 10+ | 6+ cards, **slam interest** |
| 3NT | 10-15 | To play |
| **4♣(!)** | - | **Gerber** (ace-asking) |
| 4♦(!) | - | Texas Transfer to ♥ |
| 4♥(!) | - | Texas Transfer to ♠ |
| 4♥/4♠ | - | To play (without Texas) |
| 4NT | 16-17 | **Quantitative** slam invite |
| 5NT | 20-21 | Quantitative grand slam invite |
| 6NT | 18-19 | To play |

### Stayman Convention (2♣)

**Purpose:** Find a 4-4 major fit.

**Opener's Rebids:**
| Rebid | Meaning |
|-------|---------|
| 2♦ | No 4-card major |
| 2♥ | 4+♥ (may also have 4♠) |
| 2♠ | 4♠, denies 4♥ |

**Responder's Continuations after 2♦ (no major):**
| Rebid | HCP | Meaning |
|-------|-----|---------|
| 2♥/2♠ | 8-9 | 5-card suit, invitational |
| 2NT | 8-9 | Invitational, balanced |
| 3♣/3♦ | 10+ | 5+ cards, game forcing |
| 3♥/3♠ | 10+ | 5+ cards, game forcing |
| 3NT | 10-15 | To play |
| 4♥/4♠ | 10+ | 6+ cards, to play |

**Weak Stayman:** May use with very weak hand seeking better partscore (e.g., 4-4 majors, short minor).

### Jacoby Transfers

**Mechanics:**
- 2♦ = Transfer to ♥ (opener bids 2♥)
- 2♥ = Transfer to ♠ (opener bids 2♠)

**Super-Accept:** Opener may jump to 3♥/3♠ with 4-card support and maximum (17 HCP).

**Responder's Continuations after 1NT - 2♦ - 2♥:**
| Rebid | HCP | Meaning |
|-------|-----|---------|
| PASS | 0-7 | Weak, to play |
| 2♠ | 8-9 | 5♥-4♠, invitational |
| 2NT | 8-9 | 5♥, invitational (choice of games) |
| 3♣/3♦ | 10+ | 4+ cards, second suit, game forcing |
| 3♥ | 8-9 | 6+♥, invitational |
| 3♠ | 10+ | 5-5, game forcing |
| 3NT | 10-15 | 5♥, choice of games |
| 4♥ | 10+ | 6+♥, to play |
| 4NT | 16+ | RKCB for hearts |

### Interference Over 1NT

| Opponent's Bid | Your Actions |
|----------------|--------------|
| Double | Systems ON (Stayman, transfers) |
| 2♣ | Double = Stayman; other systems ON |
| 2♦ | Double = transfer to ♥; 3♦ = Stayman |
| 2♥ or higher | Cuebid = Stayman (GF); other bids natural |

## Responses to Major Openings

Opening 1♥/1♠ shows **13-22 HCP, 5+ cards**.

### Response Point Ranges

| Points | Category |
|--------|----------|
| 0-5 | Pass (usually) |
| 6-10 | Minimum |
| 10-12 | Invitational |
| 13+ | Game forcing |
| 19+ | Slam interest |

### Response Summary (to 1♥/1♠)

| Response | HCP | Meaning |
|----------|-----|---------|
| PASS | 0-5 | Weak |
| 1♠ (over 1♥) | 6+ | 4+♠, forcing one round |
| 1NT | 6-10 | No fit, no suit at 1-level, may be passed |
| 2♣ | 10+ | 4+♣, promises rebid |
| 2♦ | 10+ | 4+♦, promises rebid |
| 2♥ (over 1♠) | 10+ | **5+♥**, promises rebid |
| **2♥/2♠ (raise)** | 6-10 | **3+ trump support** |
| **2NT(!)** | 13+ | **Jacoby 2NT** - 4+ trump, game forcing |
| 3♣/3♦ | 10-12 | 5+ cards, invitational |
| **3♥/3♠ (jump)** | 10-12 | **Limit raise**, 3-4 trump |
| 3NT | 15-17 | Balanced, 2-card support |
| **4♥/4♠ (jump)** | 6-10 | **5+ trump, distributional**, preemptive |
| Jump shift | 19+ | 4+ cards, slam interest |

### Jacoby 2NT Convention

**2NT response to 1♥/1♠** = Forcing raise with **4+ trump support** and **13+ support points**.

**Opener's Rebids after Jacoby 2NT:**
| Rebid | Meaning |
|-------|---------|
| 3♣/3♦/3♥ (new suit) | **Singleton or void** in that suit |
| 3♠ (over 1♥) | **Singleton or void** in spades |
| 3 of trump | **Maximum** (15-17), no shortness |
| 4 of trump | **Minimum** (13-14), no shortness |
| 3NT | 15-17 HCP, semi-balanced, no shortness |
| 4 of new suit | **5+ card side suit** |

### Limit Raise (Jump to 3-Major)

Shows 10-12 support points with 3-4 trump.

**Opener's Actions:**
| Opener's HCP | Action |
|--------------|--------|
| 13-14 | Pass |
| 15-17 | Bid game |
| 18+ | Explore slam |

## Responses to Minor Openings

Opening 1♣ shows **13-22 HCP, 3+♣**.
Opening 1♦ shows **13-22 HCP, 3+♦** (usually 4+).

### Response Summary (to 1♣/1♦)

| Response | HCP | Meaning |
|----------|-----|---------|
| 1♦ (over 1♣) | 6+ | 4+♦, forcing |
| 1♥ | 6+ | 4+♥, forcing |
| 1♠ | 6+ | 4+♠, forcing |
| 1NT | 6-10 | No 4-card major, not forcing |
| 2♣ (over 1♦) | 10+ | 4+♣, forcing |
| 2♦/3♣ (raise) | 6-10 | Support |
| 2♥/2♠ | 10+ | 5+ cards, forcing |
| **2NT** | 11-12 | Balanced, no 4-card major, invitational |
| 3♦/4♣ (jump) | 10-12 | Limit raise |
| **3NT** | 13-15 | Balanced, no 4-card major, to play |
| Jump shift | 19+ | 5+ cards, slam interest |

### Bidding Up the Line

With two 4-card suits, respond in the **lower-ranking** suit first (up the line) to avoid missing a fit.

**Example:** With 4♥ and 4♠, respond 1♥ to 1♣/1♦.

## Opener's Rebids

### Strength Categories

| Category | HCP | Description |
|----------|-----|-------------|
| Minimum | 13-15 | Cheapest rebid |
| Medium | 16-18 | Jump or reverse |
| Maximum | 19-21 | Double jump or jump shift |

### After 1-Level Response

| Rebid | HCP | Meaning |
|-------|-----|---------|
| 1NT | 12-14 | Balanced, no fit for responder |
| 2NT | 18-19 | Balanced |
| 3NT | 19-20 | Balanced, stoppers |
| Raise of responder | 13-16 | 4-card support |
| Jump raise | 17-19 | 4-card support |
| Rebid own suit | 13-16 | 6+ cards |
| Jump rebid own suit | 16-18 | 6+ good cards |
| New suit (non-reverse) | 13-18 | 4+ cards |
| **Reverse** | 17+ | New suit higher than opening at 2-level |
| **Jump shift** | 19+ | 4+ cards, forcing to game |

### After 2-Level Response (2/1)

In SAYC, a 2-level response in a new suit is **forcing one round** (not game-forcing like 2/1 GF system).

| Rebid | Meaning |
|-------|---------|
| 2 of own suit | Minimum, 6+ cards, or no better bid |
| 2NT | 12-14, balanced |
| 3NT | 15-17, balanced, stoppers |
| Raise | Support for responder |
| New suit | Natural, 4+ cards |

### After 1NT Response

| Rebid | HCP | Meaning |
|-------|-----|---------|
| PASS | 12-14 | Balanced minimum |
| 2 of lower suit | 12-16 | 4+ cards in new suit |
| 2 of own suit | 12-16 | 6+ cards (rebiddable) |
| 2NT | 18-19 | Balanced |
| 3 of own suit | 16-18 | 6+ good cards |
| 3NT | 19-20 | Balanced |

### Reverse Bids

A **reverse** is a new suit at the 2-level that is **higher-ranking than the opening suit**.

**Requirements:** 17+ HCP, typically 5-4 or better shape.

**Examples:**
- 1♣ - 1♠ - 2♦ = Reverse (diamonds higher than clubs)
- 1♣ - 1♠ - 2♥ = Reverse (hearts higher than clubs)
- 1♦ - 1♠ - 2♥ = Reverse (hearts higher than diamonds)

**Note:** 1♣ - 1♦ - 1♥ - 1♠ is **natural** (not fourth suit forcing in this sequence).

## Strong 2♣ Opening

Shows **22+ HCP** or a hand with **9+ playing tricks**.

### Responses to 2♣

| Response | HCP | Meaning |
|----------|-----|---------|
| **2♦(!)** | 0-7 | Waiting/artificial (may have good hand) |
| 2♥ | 8+ | **5+ cards**, 2 of top 3 honors |
| 2♠ | 8+ | **5+ cards**, 2 of top 3 honors |
| 2NT | 8+ | Balanced |
| 3♣ | 8+ | **5+ cards**, 2 of top 3 honors |
| 3♦ | 8+ | **5+ cards**, 2 of top 3 honors |

### Opener's Rebids After 2♣ - 2♦

| Rebid | Meaning |
|-------|---------|
| 2♥/2♠ | 5+ cards, forcing (may stop below game if both minimum) |
| 2NT | **22-24 HCP**, balanced (systems on: Stayman, transfers) |
| 3♣/3♦ | 5+ cards, forcing |
| 3♥/3♠ | 6+ self-sufficient suit, game forcing |
| 3NT | **25-27 HCP**, balanced |

### After 2♣ - 2♦ - 2NT (22-24)

All standard 2NT responses apply:
- 3♣ = Stayman
- 3♦ = Transfer to ♥
- 3♥ = Transfer to ♠
- 3NT = To play
- 4♣ = Gerber

### Second Negative

After 2♣ - 2♦ - 2♥/2♠, responder can bid **3♣** as a second negative (very weak hand). This keeps bidding low while showing weakness.

## Weak Two Bids

2♦/2♥/2♠ show **5-11 HCP** with a **6-card suit**.

### Requirements

- 6-card suit of reasonable quality (typically 2+ honors)
- No void
- No outside 4-card major
- Not suitable for 1-level opening

### Responses to Weak Twos

| Response | Meaning |
|----------|---------|
| PASS | Weak, no game interest |
| **2NT(!)** | **Forcing inquiry** (asks for feature or strength) |
| Raise to 3 | Invitational or preemptive |
| Raise to 4 | To play (often preemptive) |
| New suit | **Forcing one round** (RONF), 5+ cards |
| 3NT | To play |

### Responses to 2NT Inquiry

**Opener rebids:**
| Rebid | Meaning |
|-------|---------|
| 3 of suit (rebid) | **Minimum** (5-8), no feature |
| New suit | **Feature** (outside A or protected K) |
| 3NT | **Maximum** with solid suit (AKQxxx) |

| 3♣ | 7+ | 5-11 |
| 3♦ | 7+ | 5-11 |
| 3♥ | 7+ | 5-11 |

- Non-vulnerable vs. vulnerable: within 3 tricks of contract
- Equal vulnerability: within 2 tricks

## Slam Bidding

### Blackwood (4NT)

**After suit agreement**, 4NT asks for aces.

| Response | Aces |
|----------|------|
| 5♣ | 0 or 4 |
| 5♦ | 1 |
| 5♥ | 2 |
| 5♠ | 3 |

**5NT (after 4NT):** Asks for kings (guarantees all 4 aces, invites grand slam).

### Gerber (4♣)

**In NT auctions**, 4♣ asks for aces.

| Response | Aces |
|----------|------|
| 4♦ | 0 or 4 |
| 4♥ | 1 |
| 4♠ | 2 |
| 4NT | 3 |

**5♣ (after Gerber):** Asks for kings.

### Grand Slam Force (5NT)

A **jump to 5NT** (not after 4NT) asks partner to bid 7 with **2 of the top 3 trump honors** (A, K, Q).

### Cuebidding

After trump agreement, a new suit below game shows **control** (Ace or void, later King or singleton) and slam interest.

### Interference Over Blackwood/Gerber

**DOPI (Double = 0, Pass = 1):**
- Double = 0 aces
- Pass = 1 ace
- Next step = 2 aces, etc.

**ROPI (Redouble = 0, Pass = 1):**
Used when opponent doubles (not overcalls).

## Competitive Bidding

### Negative Doubles

**Range:** Through **3♠** (some partnerships play higher).

**Shows:** Values in unbid suits, especially 4-card majors.

| Auction | Double Shows |
|---------|--------------|
| 1♣ - (1♦) - X | 4-4 in majors (or better) |
| 1♣ - (1♥) - X | 4+♠ |
| 1♣ - (1♠) - X | 4+♥ |
| 1♦ - (1♥) - X | 4+♠ (1♠ = 5+) |
| 1♦ - (1♠) - X | 4+♥ |
| 1♥ - (1♠) - X | Support for minors |
| 1♥ - (2♣) - X | 4+♠ |

**HCP Required:**
| Level | Minimum HCP |
|-------|-------------|
| 1-level | 6+ |
| 2-level | 8+ |
| 3-level | 10+ |

### After Opponent's Takeout Double

| Action | Meaning |
|--------|---------|
| 1-level suit | Forcing one round |
| 2-level suit | Non-forcing, 6+ cards, <10 HCP |
| Jump raise | **Preemptive** |
| **2NT (Jordan)** | **Limit raise or better** (3+ trump over major, 5+ over minor) |
| Redouble | **10+ HCP**, often denies fit |
| Jump shift | Preemptive |

### Cuebid of Opponent's Suit

**Direct cuebid** (responder cuebids overcaller's suit):
- Game forcing
- Often asking for stopper for 3NT
- May show limit raise or better

### Free Bids

When opponent overcalls, responder's bids promise extra values (not just responding to "save" the auction).

## Overcalls

### Simple Overcalls

| Level | HCP | Suit Length |
|-------|-----|-------------|
| 1-level | 8-16 | 5+ cards (rarely strong 4) |
| 2-level | 10-16 | 5+ cards |

**With stronger hands** (17+): Double first, then bid suit.

### Responses to Overcalls

| Response | Meaning |
|----------|---------|
| Simple raise | 6-11, 3+ trump |
| Jump raise | **Preemptive**, 4+ trump |
| Jump to game | **Preemptive**, 5+ trump |
| 1NT | 9-11, balanced, stopper |
| 2NT (non-jump) | 11-13 |
| 2NT (jump) | 13-15 |
| 3NT | 15-16 |
| New suit | Non-forcing, usually denies fit |
| Cuebid | **Limit raise or better** |

### 1NT Overcall

Shows **15-18 HCP**, balanced, with **stopper(s)** in opponent's suit.

**Responses:** All systems on (Stayman, transfers).

**Balancing 1NT:** 11-15 HCP (lighter in fourth seat).

### Jump Overcalls

**Preemptive**, same requirements as opening preempt at that level.

### 2NT Overcall (Unusual NT)

Shows **5-5 or better** in the **two lowest unbid suits**.

| Opponent Opens | 2NT Shows |
|----------------|-----------|
| 1♣ | ♦ + ♥ |
| 1♦ | ♣ + ♥ |
| 1♥ | ♣ + ♦ |
| 1♠ | ♣ + ♦ |

**Strength:** Either weak (<opening) OR 17+ planning further action.

### Michaels Cuebid

**Direct cuebid** of opponent's opening shows 5-5 two-suiter:

| Opponent Opens | Cuebid Shows |
|----------------|--------------|
| 1♣ | Both majors (5-5) |
| 1♦ | Both majors (5-5) |
| 1♥ | ♠ + minor (5-5) |
| 1♠ | ♥ + minor (5-5) |

**Strength:** Same as Unusual NT (weak or strong).

**Responses to Michaels over major:**
- 2NT = Asks for minor
- 3 of known major = Weak preference
- 4 of known major = To play

## Doubles

### Takeout Doubles

Made over opponent's **partscore opening** with:
- Support for unbid suits, OR
- Strong hand (17+) planning to bid own suit

**Typical shape:** 4-4-4-1 or 5-4-3-1 with shortness in opponent's suit.

### Responses to Takeout Double

| Response | HCP | Meaning |
|----------|-----|---------|
| Cheapest suit | 0-9 | Forced response |
| Jump in suit | 10-12 | Invitational, non-forcing |
| 1NT | 6-10 | Balanced, stopper |
| 2NT | 10-12 | Balanced, stopper |
| Cuebid | 13+ | Forcing, asks for clarification |
| 3NT | 13-16 | To play, stoppers |
| Double jump | 0-6 | Preemptive |
| PASS | 5+ | **Penalty** (good cards in opponent's suit) |

### Penalty Double of 1NT

Shows similar strength/shape to 1NT opener (15-17, balanced).

Partner should:
- Pass with decent hand
- Pull with weak distributional hand

### Balancing Doubles

Made in **fourth seat** when it would pass out.

Lighter requirements (about 3 HCP less than direct seat).

| Action | Meaning |
|--------|---------|
| Double | Takeout |
| 1NT | 11-15, balanced |
| Jump 2NT | 18-20, balanced |
| Suit bid | 8-14 |
| Jump suit | Opening hand |

## Defensive Leads and Signals

### Opening Leads

**Against Suit Contracts:**
| Holding | Lead |
|---------|------|
| AKx(x) | A (or K by agreement) |
| KQx(x) | K |
| QJx(x) | Q |
| JT9(x) | J |
| T98(x) | T |
| AKJx(x) | A, then K |
| Doubleton | Top card |
| xxx | Lowest (or MUD) |
| xxxx | 4th best |

**Against NT Contracts:**
| Holding | Lead |
|---------|------|
| AKJxx+ | A (asks for unblock) |
| KQJx(x) | K |
| KQTx(x) | K |
| KQ9x(x) | **4th best** (low from touching honors) |
| QJTx(x) | Q |
| JT9x(x) | J |
| xxxx(x) | **4th best** |

### Signals

**Attitude (when partner leads):**
- **High** = Encouraging (want suit continued)
- **Low** = Discouraging

**Count (when declarer leads):**
- **High-low** = **Even** number of cards
- **Low-high** = **Odd** number of cards

**Suit Preference (when obvious):**
- High card = Higher-ranking suit
- Low card = Lower-ranking suit
