# Bridge Docs

Documentation for our bridge bidding system and conventions.

## System Overview

Our partnership plays **SAYC (Standard American Yellow Card)**, the official ACBL system:

- 5-card majors
- 15-17 NT
- Strong artificial 2♣ (22+)
- Stayman and Jacoby Transfers
- Blackwood 4NT

**SAYC** is the most widely used bidding system worldwide, serving as the default for online play (BBO) and ACBL tournaments.
