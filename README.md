# Bridge Docs

Documentation for our bridge bidding system and conventions.

## System Overview

This is a **Standard Natural bidding system** based on SAYC (Standard American Yellow Card) conventions:

- 5-card majors
- 15-17 NT
- Strong artificial 2♣ (22+)
- Stayman and Jacoby Transfers
- Blackwood 4NT
